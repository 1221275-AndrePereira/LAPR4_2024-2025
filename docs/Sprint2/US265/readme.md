# US265 - Generate a simulation report


## 1. Requirements Engineering

### 1.1. User Story Description

As a system user, I want to receive a summary of the simulation results so that I can determine if the
figure is safe to use.

### 1.3. Acceptance Criteria

* **AC1:** The system must generate a report and store it in a file.
* **AC2:** The report should include the total number of drones and their execution status.
* **AC3:** If collisions occur, the report must list timestamps and positions.
* **AC4:** The report should indicate whether the figure passed or failed validation.
